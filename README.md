## Installation

1. Clone this repository and move into the folder:
   ```bash
   git clone <repo-url>
   cd warden-test-one
   ```
2. Install Dependencies
   ```bash
   npm i
   npm run prisma:gen
   ```
3. Copy Environment File
   ```bash
   cp .env.example .env
   ```
4. Start the development server
   ```bash
   npm run dev
   ```
   open `http://localhost:5000` you should see "Warden Weather Test: OK"
